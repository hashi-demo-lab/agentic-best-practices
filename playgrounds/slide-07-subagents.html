Subagents: Custom Specialists

Markdown files that define specialized AI assistants with scoped tools, models, and permissions

Definition — markdown files with YAML frontmatter defining custom prompts, tool access, model selection, and permission boundaries
Scoped capabilities — each agent gets only the tools it needs; disallowedTools blocks dangerous operations at the boundary
Context extension — isolated context windows extend total capacity beyond single-agent limits; agents hand off via disk artifacts, keeping each window focused
Built-in types — Explore (Haiku, read-only), Plan (read-only), General-purpose (all tools), or define your own custom specialists
