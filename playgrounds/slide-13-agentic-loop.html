Agentic Loop

Validation-driven cycle: write tests, write code, validate, deploy

Write tests — test-writer agent creates .tftest.hcl from design.md with 1:1 assertion mapping before any implementation
terraform init -backend=false terraform validate
Write code — developer agent implements one checklist item at a time; tests provide the feedback loop for iteration
terraform validate terraform test
Validate — full pipeline runs fmt, validate, test, tflint, trivy; validator agent scores across 6 quality dimensions
trivy config . terraform-docs
Deploy — checkpoint commits on each passing iteration; PR created for human review; HCP Terraform runs plan/apply
